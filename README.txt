Restaurant

Program Author: Steven Kong

Purpose: Simulate an online restaurant supporting user login and registration

The server supports:
/: homepage
/users: view a list of public users
/login: go to the login page
/users/someID: view a user with id someID and have access to changing the user's privacy settings
/register: register as a new user

To run: Extract the zip file contents
Run mongo and mongod in terminal
Use "npm init"
Use "npm install"
Use "node database-initializer2.js" to initialize the database (Ignore the name, had a copy that could not be deleted and would not save)
Use "npm start" to start the server
